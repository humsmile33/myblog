# Next.js로 블로그 만들기

Next.js는 React 기반의 프레임워크로, **SSG(Static Site Generation)**를 강력하게 지원합니다.

## 왜 Next.js인가?

1. **성능**: 빌드 시점에 HTML을 생성하므로 로딩 속도가 매우 빠릅니다.
2. **SEO**: 검색 엔진 최적화에 유리합니다.
3. **개발 경험**: Fast Refresh 등 훌륭한 개발 도구를 제공합니다.

```javascript
console.log("Hello, Next.js!");
```

이 글에서는 Next.js의 기본 설정부터 배포까지 다룹니다.
